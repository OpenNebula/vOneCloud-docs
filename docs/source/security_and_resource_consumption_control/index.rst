vOneCloud ships with several authentication plugins which can be configured to pull user data from existingauthentication backends. 

It also implements a powerful permissions, quotas and ACLs mechanisms to control which users and groups are allowed to use which physical and virtual resources, keeping a record of the compsumption of these resources as well as monitoring their state periodically.

Take control of your cloud infrastructure!
